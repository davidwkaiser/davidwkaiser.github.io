## How to run this app locally

### download the repo
`git clone git@ghe.fyiblue.com:I371030/DialogFlowRepo.git`

### install a Flask server
`pip install Flask`

### run the server
`export FLASK_APP=bot.py`
`flask run`

### load the page
go to 'http://127.0.0.1:5000/ '
